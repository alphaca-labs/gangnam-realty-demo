## 0. 콘셉트

ChatGPT 웹 UI(chatgpt.com)를 레퍼런스로, **AI 채팅 인터페이스**를 메인으로 하되 **기능별 샘플 페이지**를 사이드바에서 접근 가능하게 구성.

### ChatGPT UI 참고 포인트:
- **사이드바(좌측):** 대화 히스토리 리스트 → 우리는 기능 메뉴로 활용
- **메인 영역(중앙):** 채팅 UI — 사용자 메시지 + AI 리치 응답
- **입력창(하단):** 큰 텍스트 입력 + 전송 버튼
- **초기 화면:** 로고 + 추천 질문 카드 (대화 시작 전)
- **클린 디자인:** 흰 배경, 충분한 여백, 타이포그래피 중심

## 2. 페이지별 상세

### 2-1. 메인 채팅 페이지 (ChatGPT 스타일)

**초기 상태 (대화 없을 때):**
```
┌─────────────────────────────────────────┐
│                                         │
│         🏛️ 강남부동산톡                  │
│     강남 부동산, 무엇이든 물어보세요      │
│                                         │
│  ┌──────────┐  ┌──────────┐             │
│  │ 압구정동   │  │ 중개수수료 │            │
│  │ 실거래가   │  │ 얼마인가요│            │
│  └──────────┘  └──────────┘             │
│  ┌──────────┐  ┌──────────┐             │
│  │ 전세사기   │  │ 거래신고  │            │
│  │ 예방 방법  │  │ 절차     │            │
│  └──────────┘  └──────────┘             │
│                                         │
│  ┌───────────────────────────────────┐  │
│  │ 메시지를 입력하세요...         [⬆] │  │
│  └───────────────────────────────────┘  │
└─────────────────────────────────────────┘
```

**대화 진행 중:**
- 사용자 메시지: 우측 정렬, 회색 배경
- AI 응답: 좌측 정렬, 아바타 + 리치 카드
- 리치 응답에 차트/테이블/가이드 카드 포함
- 타이핑 애니메이션 (ChatGPT처럼 글자 하나씩)

**하드코딩 시나리오 5개:**
1. "압구정동 실거래가" → 라인차트 + 거래내역 테이블
2. "중개수수료 얼마" / "수수료 계산" → 인라인 계산기 카드
3. "전세사기 예방" → 체크리스트 가이드 카드
4. "거래신고 방법" → 스텝 바이 스텝 절차 카드
5. 그 외 → "현재 데모 버전입니다. 다음 질문을 시도해보세요:" + 추천

### 2-2. 기능 ① 중개수수료 계산기

독립 페이지. 실제로 계산 동작하게 구현.
- 거래유형(매매/전세/월세) 탭
- 금액 입력 + 퀵버튼 [1억][3억][5억][10억]
- 실시간 계산 결과 (요율, 수수료, 부가세, 총 비용)
- 서울시 조례 기준 요율표 아코디언
- 월세: 보증금 + 월세 입력 → 환산보증금 자동 계산

### 2-3. 기능 ② 중개대상물 확인서 도우미

스텝 위저드 UI (목업).
- Step 1: 기본정보 (소재지, 면적, 용도)
- Step 2: 권리관계 (소유자, 저당권, 임차인)
- Step 3: 시설현황 (방/욕실/주차)
- Step 4: 확인사항 (실제 입력은 disabled — "본 서비스 오픈 시 사용 가능" 안내)
- 미리보기: 견본 양식에 샘플 데이터 표시

### 2-4. 기능 ③ 전자계약 가이드

- 매매/전세/월세 탭 전환
- 각 유형별 계약 절차 플로우차트 (타임라인 UI)
- 필요 서류 체크리스트 (인터랙티브 체크)
- 샘플 계약서 미리보기 (CSS로 계약서 느낌 렌더링)
- 특약 예시 카드 모음
- 외부 링크: 부동산거래전자계약시스템

### 2-5. 기능 ④ 웹버전 타입별 출력 예시

3-column 레이아웃 (모바일: 1-column):
- **이미지:** 샘플 갤러리 (단지 외관, 인포그래픽, 차트 이미지)
- **서류:** PDF/문서 뷰어 목업 (등기부등본, 건축물대장 견본)
- **문서·양식 가이드:** 거래신고서 작성 가이드, 양식 다운로드 버튼

각 타입별 "카카오톡에서는 → 웹에서는" 비교 배너 포함.

### 2-6. 기능 ⑤ 실거래가 시각화 대시보드

- 동/단지 필터 드롭다운
- 인터랙티브 라인 차트 (Recharts) — 기간별 가격 추이
- 거래 내역 테이블 (정렬·검색)
- 통계 카드 (평균/최고/최저/건수)
- 목데이터: 강남구 주요 단지 50건+

### 2-7. 기능 ⑥ 강남구 부동산 지도

- 지도 UI (정적 지도 이미지 또는 Kakao Maps embed)
- 더미 마커 10~15개 (주요 단지 위치)
- 마커 클릭 → 팝업 (단지명, 평균가, 면적)
- 필터 패널: 가격대 슬라이더(목업), 유형 체크박스
- 동별 색상 구분 (히트맵 느낌)

### 2-8. 기능 ⑦ 민원 절차 가이드 포털

4개 카테고리 카드:
1. **부동산 거래신고** — 5단계 스텝퍼
2. **토지거래허가** — 신청→심사→허가 플로우
3. **중개업 개설** — 자격→등록→개설 절차
4. **전세사기 예방** — 8개 항목 체크리스트 + 주의사항

각 카테고리 클릭 → 상세 스텝 가이드 아코디언 확장.

### 2-9. 기능 ⑧ AI 상담 고도화 데모

기존 챗봇 vs 웹 AI 상담 비교 데모:
- 좌: "기존 챗봇" (단순 텍스트 응답 시뮬레이션)
- 우: "AI 상담 고도화" (차트+테이블+가이드 리치 응답)
- 같은 질문에 대한 양쪽 응답 비교
- "이전/다음" 버튼으로 시나리오 전환

## 4. 기술 스택

| 항목 | 선택 |
|------|------|
| 프레임워크 | Next.js 15 (App Router) |
| 스타일링 | Tailwind CSS |
| UI | shadcn/ui |
| 차트 | Recharts |
| 아이콘 | Lucide React |
| 애니메이션 | Framer Motion / CSS transitions |
| 폰트 | Pretendard (CDN) |
| 배포 | GitHub Pages (static export) |

### Static Export 주의사항
- `output: 'export'` in next.config
- `trailingSlash: true`
- `basePath: '/gangnam-realty-demo'`
- 이미지: next/image → unoptimized: true
- 동적 라우트 → generateStaticParams
